# Spring Batch Transactions

Project to demonstrate how Spring Batch manages transactions within a chunk.
The project has a job that reads a file of users and inserts them into the database.

Technologies:
 - Java 17
 - Springboot 3.3.3
 - Spring Batch 5.0
 - H2 Databse
 - MySql
